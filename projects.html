Ministère de l'eau et de l'énergie

Application web pour la monographie hydraulique en milieu rural

La fiche de la monographie hydraulique est formulaire d’une dizaine de pages permettant d’enregistrer les informations sur les types et les sources d’adduction en eau potable. L'application développée à cet effet éffectue les taches suivantes:

  • Enregistre le profl l’utilisateur
  • Enregistre les informations locales (coordonnées GPS, district, département et region)
  • Enregistre la source d’adduction (stockage, production, maintenance, mode de financement, …)
  • Propose une recherche rapide pour consulter les données enregistrer
  • Propose un espace convivial pour les utilisateurs.

Association locale (Jeunes dynamiques de Yaoundé)

Application web pour la réunion

Jeunes Dynamiques Baletet de Yaoundé est une association de personnes qui comporte en son sein une petite banque fnancée par les membres. J’ai développé une application pour la gestion de cette association. Cette application comporte deux sous applications :

  • Une application pour la gestion des membres (gestion de la vie d’un membre dans l’association)
  • Une applications pour la gestion de la banque (gestion des dettes et des emprunts, traitement des bilans financiers)
